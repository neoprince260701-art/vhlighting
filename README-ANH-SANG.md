# Thuộc tính Ánh sáng

## 1. Chạy SQL một lần
Trong Supabase: SQL Editor → New query → mở file `supabase/migration_product_light_color.sql` → dán nội dung → Run.

## 2. Cập nhật website
Upload toàn bộ code lên GitHub. Vercel sẽ tự deploy.

## 3. Cách dùng
- Vào Sản phẩm → Thêm sản phẩm.
- Mục Ánh sáng là không bắt buộc.
- Chọn Trắng, Vàng, Trung tính, Đổi 3 màu, RGB, RGBIC hoặc để trống.
- Danh sách sản phẩm và ô tìm kiếm bán hàng sẽ hiển thị thuộc tính này.

## 4. Nhập Excel
File mẫu có thêm cột `Ánh sáng`.
- Có ánh sáng: điền giá trị.
- Không áp dụng: để trống.
